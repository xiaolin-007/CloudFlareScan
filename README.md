## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| 🚀 **高效扫描** | 自动从 CloudFlare 官方 IP 段生成待测 IP 地址 |
| 📊 **双模式测速** | 完全测速 + 地区测速，满足不同需求 |
| 🌍 **全球覆盖** | 内置全球大部分机场 IATA 代码映射，自动识别地区 |
| ⚡ **异步处理** | 支持高并发测试，快速获取结果 |
| 📋 **一键复制** | 双击表格单元格即可复制内容 |
| 📈 **实时统计** | 显示扫描进度、速度 |

## 🚀 快速开始

### Windows

1. 下载 [最新版本](https://github.com/xiaolin-007/CloudFlareScan/releases)
2. 解压并运行 `CloudFlareScan.exe`

### macOS

1. 下载 [最新版本](https://github.com/xiaolin-007/CloudFlareScan/releases)
2. 如果遇到安全提示，在 **系统设置 → 隐私与安全性** 中选择「仍要打开」
3. 或在终端执行以下命令解除限制：

```bash
# 允许从任何来源安装应用
sudo spctl --master-disable
```

### Android

1. 下载 APK 文件
2. 直接安装即可

---

## 📖 使用说明

1. 启动应用后，自动加载 CloudFlare IP 段
2. 选择测速模式：
   - **完全测速**: 测试所有 IP 的完整速度
   - **地区测速**: 按地区筛选并测试
3. 点击开始扫描
4. 双击结果可复制 IP 地址
